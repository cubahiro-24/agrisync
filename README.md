# Agrisync — Site vitrine (demo)

Ceci est une version minimale du site vitrine "Agrisync" construite en React + Vite. Le contenu est en français et présente le problème, la solution et les fonctionnalités principales : diagnostic de maladies par IA et irrigation intelligente via IoT.

Prérequis
- Node.js (>=16)

Installation et démarrage

```bash
# depuis le dossier du projet
npm install
npm run dev
```

Build

```bash
npm run build
npm run preview
```

Contenu ajouté
- `index.html` — point d'entrée
- `src/` — code React (App + composants)
- `package.json`, `vite.config.js` — configuration
